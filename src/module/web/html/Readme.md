# HTML5

> `HTML5`是用来描述网页的一种语言，被称为`超文本标记语言`。

> 用`HTML5`编写的文件，后缀以`.html`结尾。

> `HTML`是一种标记语言，标记语言是一套标记标签。

> 标签是由尖括号包围的关键字，例如：`<html>`

> 标签有两种表现形式：

> 1. 双标签。例如：`<html></html>`

> 2. 单标签。例如：`<br/>`

## DOCTYPE

+ `DOCTYPE`是`document type`(文档类型)的缩写。
+ `<!DOCTYPE html>`是`HTML5`的声明，位于文档的最前面，处于标签之前。
+ `DOCTYPE`声明时网页必备的组成部分，避免浏览器的怪异模式。

```html
<!DOCTYPE html>
`
