    # VuePress 介绍

    # VuePress 是什么？

    • VuePress 是一个极简静态网站生成器，包含了 Vue 驱动的主题系统和插件API；
    • VuePress 是为书写技术文档而优化的默认主题。

    # 为什么使用 VuePress？

    • VuePress 专注在以内容为中心的静态网站上，同时提供了一些为技术文档定制的开箱即用的特性；
    • VuePress 对 Markdown 支持友好，配置灵活方便

    # 怎么使用 VuePress？

    1. 创建并进入一个新目录
      mkdir vuepress-starter && cd vuepress-starter
    1. 初始化项目环境
      npm init --y
    1. 安装 VuePress
      npm i -D vuepress
    1. 创建一篇文章
      mkdir docs && echo "# Hello VuePress" > docs/README.md
    1. 在package.json中修改scripts
      "scripts":{
      "dev":"vuepress dev docs --no-cache --temp .temp",
  }
    1. 运行项目
      yarn dev

    VuePress 会在http://localhost:8080启动一个热重载的开发服务器。

    # VuePress 插件

    # back-to-top

    可快速返回页面顶部

    # 安装

    yarn add -D @vuepress/plugin-back-to-top

    # 使用

    在.vuepress/config.js中添加plugins:

    module.exports = {
  plugin: ['@vuepress/back-to-top']
}

    # last-updated

    如果你对时间格式没有特殊需求，按照默认即可，不需安装该插件。
